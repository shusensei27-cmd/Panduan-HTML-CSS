1. Layout Halaman (CSS Grid)

grid-template-areas: Mendefinisikan layout visual halaman. Setiap baris dalam string = 1 baris grid. Nama area = grid-area pada elemen.
<header>...</header> <nav>...</nav> <main>...</main> <aside>...</aside> <footer>...</footer> CSS: .grid-layout { display: grid; grid-template: "hdr hdr" auto "nav main" 1fr "ftr ftr" auto / 220px 1fr; gap: 0; min-height: 100vh; }

2. Navigasi Responsif (Flexbox)

Navbar dengan Flexbox: justify-content:space-between untuk logo di kiri dan links di kanan. align-items:center untuk vertikal tengah. transition untuk efek hover.
<nav class="navbar"> <a href="#" class="logo">MySite</a> <ul class="nav-links"> <li><a href="#">Home</a></li> <li><a href="#">About</a></li> </ul> </nav> CSS: .navbar { display: flex; justify-content: space-between; align-items: center; padding: 1rem 2rem; }

3. Hero Section (Flexbox + Gradien + Animasi)

Hero: display:flex + min-height untuk full height. linear-gradient untuk background. @keyframes untuk animasi. clamp() untuk font-size responsif.
<section class="hero"> <div class="hero-content"> <h1>Belajar HTML & CSS</h1> <p>Dari nol hingga mahir...</p> <a href="#" class="btn">Mulai</a> </div> </section> CSS: .hero { display: flex; align-items: center; min-height: 60vh; background: linear-gradient(135deg, #2563eb, #7c3aed); color: white; }

4. Cards Grid (Grid auto-fit + Hover)

repeat(auto-fit, minmax(280px, 1fr)): Otomatis menyesuaikan jumlah kolom. Card dengan border, border-radius, box-shadow, dan hover translateY untuk efek angkat.
<div class="card-grid"> <div class="card"><h3>HTML</h3><p>...</p></div> <div class="card"><h3>CSS</h3><p>...</p></div> <div class="card"><h3>JS</h3><p>...</p></div> </div> CSS: .card-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 1rem; } .card { background: var(--bg); border: 1px solid var(--bd); border-radius: 12px; padding: 1.5rem; transition: all 0.3s; } .card:hover { transform: translateY(-4px); }

5. Form dengan Validasi

Form: type=text/email/tel/textarea/checkbox/radio. Validasi HTML5: required, pattern, minlength. Pseudo-class :user-valid/:user-invalid untuk styling.
<form> <label>Nama *</label> <input type="text" required> <label>Email *</label> <input type="email" required> <label>Pesan</label> <textarea></textarea> <button>Kirim</button> </form>

6. Animasi CSS (@keyframes)

@keyframes: Mendefinisikan animasi multi-tahap. animation shorthand untuk menerapkan. prefers-reduced-motion untuk aksesibilitas.
@keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-20px); } } @keyframes pulse { 0%, 100% { opacity: 1; transform: scale(1); } 50% { opacity: 0.7; transform: scale(0.95); } } @keyframes spin { from { transform: rotate(0); } to { transform: rotate(360deg); } } .element { animation: bounce 1s ease infinite; }

7. Tabel dengan Stripped Rows

CSS Table: border-collapse:collapse untuk border rapat. :nth-child(even) untuk stripped rows. :hover untuk highlight baris.

8. Variasi Tombol (Button Styles)

Button variants: Primary, Outline, Ghost, Disabled. Masing-masing hover state beda. transition untuk animasi halus.
.btn-primary { background: #2563eb; color: white; border: none; padding: 0.5rem 1.5rem; border-radius: 8px; transition: all 0.3s; cursor: pointer; } .btn-primary:hover { background: #1d4ed8; transform: translateY(-2px); }

9. Flexbox vs Grid

Flexbox = 1 dimensi (baris ATAU kolom). Grid = 2 dimensi (baris DAN kolom). Pilih sesuai kebutuhan!
<!-- Flexbox 1D --> <div class="flex"> <span>Item 1</span> <span>Item 2</span> <span>Item 3</span> </div> CSS: .flex { display: flex; gap: 1rem; } <!-- Grid 2D --> <div class="grid"> <div>A</div><div>B</div> <div>C</div><div>D</div> </div> CSS: .grid { display: grid; grid-template-columns: 1fr 1fr; gap: 1rem; }
