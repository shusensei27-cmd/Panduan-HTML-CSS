1. Struktur Dasar HTML

Setiap dokumen HTML WAJIB memiliki struktur dasar ini.

Struktur Dasar HTML5

!DOCTYPE html HARUS di baris pertama. Tanpa ini browser masuk quirks mode!

html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <title>Judul</title>
</head>
<body>
  <h1>Halo Dunia!</h1>
  <p>Paragraf pertama</p>
</body>
</html>
Preview:

Halo Dunia!

Paragraf pertama

Elemen head - Metadata

Isi head: title (WAJIB), meta, link, style, script.

html
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Belajar HTML">
  <title>Judul Halaman</title>
  <link rel="stylesheet" href="style.css">
  <style>body{font-family:Arial}</style>
</head>
Preview:

head: Berisi metadata, title, link CSS, style internal

Konten di head TIDAK tampil di browser

Elemen body - Konten Visual

body = semua konten visual. Tempatkan script di akhir body untuk performa.

html
<body>
  <header>Header</header>
  <main>
    <h1>Judul</h1>
    <p>Konten utama</p>
  </main>
  <footer>Footer</footer>
</body>

2. Elemen Semantik HTML5

Elemen semantik memberi MAKNA pada konten. Penting untuk SEO dan aksesibilitas.

header - Kepala

Bisa muncul lebih dari sekali (header global + header artikel).

html
<header>
  <h1>Website Saya</h1>
  <nav><a href="#">Beranda</a></nav>
</header>
Preview:

Website Saya

nav - Navigasi

Hanya untuk navigasi UTAMA. Gunakan aria-label jika ada lebih dari 1 nav.

html
<nav aria-label="Menu">
  <ul>
    <li><a href="/">Beranda</a></li>
    <li><a href="/blog">Blog</a></li>
  </ul>
</nav>
Preview:
  • Beranda
  • Blog

Hanya BOLEH SATU main per halaman!

html

    <p>Isi artikel utama.</p>
  </article>
</main>

Setiap section HARUS punya judul atau heading!

3. Heading h1 sampai h6

Heading untuk judul. Hanya SATU h1 per halaman! Hierarki penting untuk SEO.

Hierarki Heading

Jangan lompat hierarki (h1 ke h3). Gunakan CSS untuk ukuran, bukan heading level.

html
<h1>Judul Utama (h1)</h1>
<h2>Sub Judul (h2)</h2>
<h3>Sub-sub (h3)</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Heading dalam Section

Struktur hierarkis: h1 ke h2 ke h3. Screen reader menggunakan heading untuk navigasi.

html
<article>
  <h1>Judul Artikel</h1>
  <section>
    <h2>Pendahuluan</h2>
    <p>Teks pendahuluan...</p>
  </section>
  <section>
    <h2>Pembahasan</h2>
    <h3>Sub-topik 1</h3>
    <p>Detail sub-topik...</p>
  </section>
</article>

4. Elemen Teks dan Formatting

Elemen untuk memformat konten tulisan.

p - Paragraf

Jangan gunakan br berganda untuk jarak - pakai CSS margin.

html
<p>Ini paragraf pertama.</p>
<p>Ini paragraf kedua.</p>
Preview:

Ini paragraf pertama.

Ini paragraf kedua.

span - Inline Container

Untuk styling bagian kecil teks. Tidak punya makna semantik.

html
<p>Teks <span style="color:red">merah</span> di tengah.</p>
Preview:

Teks merah di tengah.

div - Block Container

Pilihan terakhir - usahakan pakai elemen semantik dulu.

html
<div class="card">
  <h3>Judul Card</h3>
  <p>Isi card di sini.</p>
</div>
Preview:

Judul Card

Isi card di sini.

em vs i - Miring

em = penekanan (intonasi di screen reader). i = hanya tampilan miring.

html
<p><em>Sangat</em> penting (em).</p>
<p><i>Hanya miring</i> (i).</p>
Preview:

Sangat penting - screen reader memberi intonasi.

Hanya miring - tanpa makna.

strong vs b - Tebal

strong = kepentingan tinggi. b = hanya tampilan tebal.

html
<p><strong>Peringatan!</strong> (strong)</p>
<p><b>Tebal saja</b> (b).</p>
Preview:

Peringatan! - screen reader nada tegas.

Tebal saja - tanpa makna.

mark - Highlight

Default background kuning. Untuk hasil pencarian atau kata kunci.

html
<p>Bagian <mark>paling penting</mark> di sini.</p>
Preview:

Bagian paling penting di sini.

small - Teks Kecil

Untuk disclaimer, catatan kaki, hak cipta.

html

Preview:

Rp 100.000 (belum pajak)

del dan ins - Hapus dan Sisip

del = coret (dihapus). ins = garis bawah (disisipkan).

html

Harga Rp 150rb Rp 100rb

sub = subscript (H2O). sup = superscript (x2).

html

H2O dan x2

Elemen tautan, gambar, audio, video, iframe.

a - Hyperlink

html
<a href="https://example.com">Kunjungi Example</a>
<a href="mailto:admin@test.com">Kirim Email</a>
<a href="#" target="_blank" rel="noopener">Tab Baru</a>

Atribut WAJIB: src (URL) dan alt (teks alternatif untuk aksesibilitas).

Sediakan multiple format. Browser pilih source pertama yang cocok.

html

</picture>

7. Elemen Tabel

Tabel untuk menampilkan data dalam baris dan kolom.

table - Tabel Lengkap

Struktur: thead (header), tbody (body), tfoot (footer). caption = judul tabel.

html
<table border="1">
  <caption>Data Penjualan</caption>
  <thead>
    <tr><th>Bulan</th><th>Penjualan</th></tr>
  </thead>
  <tbody>
    <tr><td>Jan</td><td>10jt</td></tr>
    <tr><td>Feb</td><td>12jt</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td><td>22jt</td></tr>
  </tfoot>
</table>

colspan dan rowspan

colspan = gabung kolom. rowspan = gabung baris. scope=col/row untuk aksesibilitas.

html
<table border="1">
  <tr>
    <th rowspan="2">Nama</th>
    <th colspan="2">Kontak</th>
  </tr>
  <tr>
    <th>Email</th><th>Telp</th>
  </tr>
  <tr>
    <td>John</td><td>john@mail.com</td><td>081234</td>
  </tr>
</table>

9. Elemen Interaktif & Multimedia Lain

Elemen interaktif HTML5 tanpa JavaScript.

details dan summary

Expand/collapse MURNI HTML! Atribut open = default terbuka.

html
<details>
  <summary>Klik untuk buka</summary>
  <p>Konten tersembunyi! Tanpa JavaScript!</p>
</details>
Preview:
Klik untuk buka

Konten tersembunyi! Tanpa JavaScript!

dialog - Modal

Method: showModal() = modal dengan overlay. show() = non-modal. close() = tutup.

html
<dialog>
  <h2>Selamat Datang!</h2>
  <p>Ini dialog modal HTML5.</p>
  <button>Tutup</button>
</dialog>
Preview:

[Klik tombol untuk membuka dialog]

canvas - Gambar JS

Dikontrol dengan JavaScript. Untuk game, grafik, visualisasi data.

html
<canvas id="c" width="200" height="100"></canvas>
<script>
  var ctx = c.getContext("2d");
  ctx.fillStyle = "blue";
  ctx.fillRect(20, 20, 150, 60);
</script>

Canvas digambar dengan JavaScript
