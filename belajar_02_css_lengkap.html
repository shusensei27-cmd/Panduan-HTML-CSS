1. CSS Selectors

Selector menentukan elemen mana yang di-styling.

Type Selector (Elemen)

Memilih SEMUA elemen dengan tag tertentu. Spesifisitas: 0,0,0,1

css
p { color: #2563eb; font-weight: 600; }
html
<p>Teks biru tebal</p><span>Span tidak berubah</span>
Preview:

Teks biru tebal

Span tidak berubah

Class Selector (.)

Class bisa dipakai berkali-kali. Spesifisitas: 0,0,1,0

css
.highlight { background: #fef3c7; padding: 0.25rem 0.5rem; border-radius: 4px; font-weight: bold; }
.card { border: 2px solid #e2e8f0; padding: 1rem; border-radius: 8px; }
html
<p class="highlight">Teks highlight</p><div class="card">Kartu dengan border</div>
Preview:

Teks highlight

Kartu dengan border

ID Selector (#)

ID harus UNIK - hanya satu per halaman! Spesifisitas: 0,1,0,0

css
#header { background: #1e293b; color: white; padding: 1rem; border-radius: 8px; }
html
<div id="header">Header dengan background gelap</div>
Preview:
Header dengan background gelap

Universal Selector (*)

Hati-hati performa! Gunakan untuk CSS reset dasar saja.

css
* { margin: 0; padding: 0; box-sizing: border-box; }
html
<div><p>Teks dalam div</p><span>Span dalam div</span></div>
Preview:

Teks dalam div

Span dalam div

Descendant (spasi) vs Child (>)

spasi = semua level turunan. > = langsung saja.

css
article p { color: #555; } /* semua p di article */
article > p { font-weight: bold; } /* p langsung anak */
html
<article>
  <p>Langsung anak (bold)</p>
  <div>
    <p>Cucu (tidak bold, tetap #555)</p>
  </div>
</article>
Preview:

Langsung anak (bold + #555)

Cucu (hanya #555, descendant saja)

Adjacent (+) & General (~) Sibling

+ = langsung setelah. ~ = semua setelah (saudara).

css
h2 + p { color: #2563eb; font-weight: bold; } /* langsung setelah h2 */
h2 ~ p { color: #64748b; } /* semua setelah h2 */
html
<h2>Judul</h2>
<p>Paragraf 1 (+ dan ~)</p>
<p>Paragraf 2 (~ saja)</p>
Preview:

Judul

Paragraf 1 (biru bold - + dan ~)

Paragraf 2 (abu - hanya ~)

Attribute Selector

^= = diawali. $= = diakhiri. *= = mengandung. i = case-insensitive.

css
a[href^="https"] { color: green; }
a[href^="mailto:"] { color: purple; font-style: italic; }
input[type="text"] { border: 2px solid blue; }
html
<a href="https://example.com">Link aman</a>
<a href="mailto:admin@test.com">Email</a>
<input type="text" value="Text field">
Preview:
Link aman (https)Email (mailto:)

:hover, :focus, :active

:hover = mouse di atas. :focus = input aktif. :active = diklik. Urutan LVHA untuk link.

css

.btn:hover { background: #1d4ed8; transform: translateY(-2px); }
.btn:active { transform: scale(0.95); }
html
<button class="btn">Hover saya!</button>

nth-child

Rumus: an+b. 2n = genap, 2n+1 = ganjil, 3n = kelipatan 3, n+4 = mulai dari 4.

css
li:nth-child(odd) { background: #f1f5f9; }
li:nth-child(even) { background: #f8fafc; }
li:first-child { font-weight: bold; }
li:last-child { border-bottom: none; }
html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
  <li>Item 4</li>
</ul>
Preview:
  • Item 1 (bold + ganjil)
  • Item 2 (genap)
  • Item 3 (ganjil)
  • Item 4 (genap + last)

:not() - Negasi

:not() = elemen yang TIDAK cocok. Bisa dirantai: :not(.a):not(.b)

css
input:not([type="submit"]) { border: 2px solid #2563eb; padding: 0.5rem; border-radius: 6px; }
html
<input type="text" value="Text field">
<input type="submit" value="Submit">

::before & ::after

WAJIB ada content: (minimal ""). ::before = sebelum konten. ::after = setelah konten.

css
.quote::before { content: "\201C"; font-size: 2em; color: #999; vertical-align: middle; }
.required::after { content: " *"; color: red; }
html
<p class="quote">Ini kutipan</p>
<label class="required">Nama</label>
Preview:

" Ini kutipan

::first-line & ::first-letter

::first-line = baris pertama (responsif). ::first-letter = drop cap effect.

css
p::first-line { font-weight: bold; font-size: 1.2em; }
p::first-letter { font-size: 3em; float: left; margin-right: 8px; color: #e74c3c; line-height: 1; }
html
<p class="demo-p">Baris pertama bold. Baris kedua normal. Lorem ipsum dolor sit amet.</p>
Preview:

Baris pertama bold. Baris kedua normal. Lorem ipsum dolor sit amet.

2. Warna & Background

Properti warna dan background untuk tampilan visual.

color - Warna Teks

Format: nama, HEX (#RRGGBB), RGB, RGBA (dengan alpha), HSL, HSLA. Pastikan kontras 4.5:1!

css
/* Format warna:
   - Nama: red, blue, green
   - HEX: #ff0000
   - RGB: rgb(255,0,0)
   - RGBA: rgba(255,0,0,0.5)
   - HSL: hsl(0,100%,50%) */
html
<p style="color:red">Merah (nama)</p>
<p style="color:#ff0000">Merah (HEX)</p>
<p style="color:rgba(255,0,0,0.5)">Merah 50% (RGBA)</p>
Preview:

Merah (nama warna)

Merah (HEX #ff0000)

Merah 50% transparan (RGBA)

background-color

Bisa: transparent, rgba(), hsla(). Jaga kontras dengan teks!

html
<div class="box">Background biru muda</div>

Background biru muda (#dbeafe)

background-image

Bisa: none, url(), linear-gradient(), radial-gradient(). Multiple backgrounds dipisah koma.

Preview:
Background Gambar

background Shorthand

Urutan: color image repeat attachment position / size.

css
background: #2563eb url("bg.jpg") no-repeat center / cover;

Gradien dengan background shorthand

linear-gradient()

Arah: to right, to bottom, 45deg, dll. Color stop: red 0%, yellow 50%, blue 100%.

css

<div class="grad">Linear to right: biru ke ungu</div>

Linear to right: biru ke ungu

radial-gradient()

Bentuk: circle, ellipse. Posisi: at top, at center.

css

Butuh -webkit-background-clip: text. color: transparent agar background terlihat melalui teks.

}

4. Box Model

Setiap elemen adalah kotak: margin + border + padding + content.

Margin - Ruang Luar

Margin Collapse: margin vertikal antar block saling collapse (nilai terbesar menang).

css
margin: 20px; /* semua sisi */
margin: 10px 20px; /* atas-bawah | kiri-kanan */
margin: 0 auto; /* tengah horizontal */
html
<div style="background:#dbeafe;padding:0.5rem">
  <div style="background:#2563eb;color:white;padding:0.5rem;margin:20px">Margin 20px</div>
</div>
Preview:
Elemen dengan margin 20px

Area biru muda = parent. Kotak = elemen dengan margin.

Padding - Ruang Dalam

Padding TIDAK collapse (tidak seperti margin). Selalu ditambahkan.

css
padding: 20px; /* semua sisi */
padding: 10px 20px; /* atas-bawah | kiri-kanan */
html
<div style="padding:20px;background:#dbeafe;border:2px solid #2563eb;border-radius:8px">Konten dengan padding 20px</div>
Preview:
Konten dengan padding 20px di semua sisi. Jarak antara teks dan border adalah padding.

Border

Per sisi: border-top, border-right, border-bottom, border-left. Style: solid, dashed, dotted, double.

css
border: 3px solid #2563eb;
border: 3px dashed #ef4444;
border: 3px dotted #10b981;
border: 3px double #7c3aed;
html
<div style="border:3px solid #2563eb;padding:0.75rem;margin-bottom:0.5rem">Solid</div>
<div style="border:3px dashed #ef4444;padding:0.75rem">Dashed</div>
Preview:
solid - garis lurus
dashed - putus-putus

border-radius - Sudut Melengkung

50% = lingkaran (jika persegi). Per sudut: border-top-left-radius, dll.

css
border-radius: 8px; /* semua sudut */
border-radius: 50%; /* lingkaran */
border-radius: 10px 20px; /* berbeda per sudut */
html
<div style="border-radius:8px;background:#2563eb;color:white;padding:0.75rem;margin-bottom:0.5rem">Radius 8px</div>

REKOMENDASI: border-box untuk SEMUA elemen. Memudahkan perhitungan lebar!

5. Display & Position

Mengontrol tampilan dan posisi elemen.

display: block vs inline vs inline-block

display:none = dihapus dari layout. visibility:hidden = masih di layout tapi tidak terlihat.

css
display: block; /* baris baru, lebar penuh */
display: inline; /* dalam baris, selebar konten */
display: inline-block; /* inline tapi bisa diatur lebar/tinggi */
html
<div style="display:block;background:#dbeafe;padding:0.5rem;margin-bottom:0.25rem">block - lebar penuh</div>
<span style="display:inline;background:#fef3c7;padding:0.25rem">inline</span>
<span style="display:inline;background:#fef3c7;padding:0.25rem">inline jg</span>
Preview:
block - lebar penuh, baris baru
inline - selebar konteninline jg

position: relative

Elemen tetap di flow. top/left/right/bottom aktif. z-index bisa digunakan.

css
position: relative;
top: 10px;
left: 20px;
/* Masih di flow. Ruang asli tetap ada. */
html
<div style="background:#dbeafe;padding:0.5rem;margin-bottom:0.25rem">Normal</div>
<div style="position:relative;top:10px;left:20px;background:#2563eb;color:white;padding:0.5rem">relative: geser 10px ke bawah, 20px ke kanan</div>
Preview:
Normal - posisi default
relative: geser 10px ke bawah, 20px ke kanan
Normal - ruang asli relative masih ada

position: absolute

DIHAPUS dari flow. Posisi relatif ke parent non-static. Jika tidak ada, relatif ke body.

css
position: absolute;
top: 10px;
right: 10px;
/* Dihapus dari flow. Relatif ke parent non-static. */
html
<div style="position:relative;background:#f1f5f9;padding:2rem;min-height:80px;border-radius:8px">
  Parent relative
  <div style="position:absolute;top:5px;right:10px;background:#ef4444;color:white;padding:0.25rem 0.5rem;border-radius:4px">absolute</div>
</div>
Preview:
Parent relative
absolute: pojok kanan atas

position: fixed

Tetap di posisi saat discroll! Relatif ke viewport, bukan parent.

css
position: fixed;
bottom: 20px;
right: 20px;
/* Tetap di viewport saat scroll */
html
<div style="background:#f1f5f9;padding:1rem;border-radius:8px">Fixed: tombol back-to-top tetap di tempat saat scroll.</div>
Preview:
Elemen fixed tetap di posisi viewport meskipun halaman di-scroll. Contoh: navbar, tombol back-to-top.

position: sticky

Normal mengikuti flow, lalu menempel di top saat scroll mencapai threshold.

css
position: sticky;
top: 0;
/* Campuran relative + fixed */
html
<div style="background:#f1f5f9;padding:1rem;border-radius:8px">Sticky: mengikuti flow lalu menempel saat scroll.</div>
Preview:
Sticky = campuran relative + fixed. Normal mengikuti flow, lalu menempel di top saat discroll. Contoh: navbar sticky.

z-index

Hanya untuk position selain static. Nilai lebih besar = lebih depan.

css

html

z:1

visible = overflow keluar. hidden = terpotong. auto = scroll jika perlu. scroll = selalu.
