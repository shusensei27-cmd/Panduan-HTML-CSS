About Platform BelajarKoding

Platform belajar basic coding untuk para Vibes Codingers yang biasanya modal copas AI, ganti warna dikit terus merasa udah jadi programmer elite 🗿

Daripada pas ditanya:
"itu flexbox kenapa bisa center?"
malah bengong melihat masa depan...

mending cari tau dulu fundamentalnya..

Website ini fokus membahas dasar dasar frontend development seperti HTML & CSS dengan cara yang lebih santai, visual, dan mudah dipahami pemula. Cocok untuk orang yang baru masuk dunia coding, suka desain web aesthetic, atau ingin mulai ngerti isi kode yang biasanya cuma dicopas dari AI, YouTube, atau GitHub.

Belajar di sini tidak cuma teori, tapi juga langsung melihat preview hasil coding agar lebih gampang memahami cara kerja setiap elemen.

Tujuan

Tujuan website ini dibuat adalah untuk membantu pemula memahami fundamental coding web sebelum terlalu jauh bergantung pada AI atau template instan.

Karena vibes coding memang seru...
tapi lebih seru lagi kalau ngerti apa yang sebenarnya terjadi di balik layar monitor..

Platform ini dibuat agar:

  • pemula bisa belajar coding secara bertahap
  • memahami struktur HTML & styling CSS
  • bisa membuat website sendiri dari nol
  • tidak hanya menjadi "copy paste engineer"
  • membangun mindset frontend developer yang lebih paham logika tampilan web

Target

Website ini ditujukan untuk:

Vibes

Player Vibes Coding baru yang lagi cari arah

Pemula

Pemula yang ingin belajar frontend development dari dasar

Design

Orang yang suka UI aesthetic dan web design

Pelajar

Pelajar atau mahasiswa yang baru masuk dunia coding

Fundamental

Orang yang ingin ngerti fundamental HTML + CSS

Lofi

Orang yang biasanya coding ditemani kopi, playlist lofi, dan existential crisis kecil jam 1 malam ☕

Materi Yang Dipelajari

📚 Page "Panduan Lengkap"

Berisi roadmap dan panduan lengkap belajar HTML + CSS dari dasar hingga memahami struktur pembuatan website modern secara bertahap.

Isi pembelajaran:

  • pengenalan web development
  • struktur HTML
  • dasar CSS
  • layout website
  • responsive design
  • mini challenge
  • roadmap belajar frontend

📖 Page "Belajar HTML"

Berisi pembelajaran dasar HTML lengkap beserta preview hasil secara langsung agar pengguna bisa melihat bentuk output dari setiap tag yang dipelajari.

Isi pembelajaran:

  • heading
  • paragraph
  • image
  • link
  • list
  • table
  • form
  • semantic HTML
  • struktur halaman web

Disertai preview visual hasil HTML secara realtime.

🎨 Page "Belajar CSS"

Berisi pembelajaran dasar CSS beserta preview styling agar pengguna memahami cara mempercantik tampilan website.

Isi pembelajaran:

  • color
  • background
  • margin & padding
  • border
  • flexbox
  • grid
  • animation
  • hover effect
  • responsive layout

Disertai preview perubahan desain secara langsung.

🚀 Page "Proyek"

Berisi pembelajaran membuat full project HTML + CSS dari awal hingga menjadi tampilan website utuh.

Isi pembelajaran:

  • membuat landing page
  • membuat navbar
  • hero section
  • card UI
  • footer
  • responsive website
  • modern UI layout

Disertai preview project dan source code yang dapat dipelajari langsung.
